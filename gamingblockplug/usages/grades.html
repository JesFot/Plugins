Description de l'ajout de grades

Utilisation: Pas d'utilisation

Les grades permettent de gerer facilement les permissions notemment pour les commandes :
/gtpa, /spectate, /world et /ghome.
Le grade est écrit à gauche du pseudo dansle chat normal.
NB: Pour des grades personnalisé ou des problèmes avec les grades actuelles, demandez un modo ou plus.
Exemple :
    Si wormsor écrit 'coucou' dans le chat et qu'il est admin, on verra :
    '[Admin]<wormsor> coucou'.

à voir : Commande /gteams, permet de voir et/ou modifier ces grades.
