Description de l'ajout de la coloration du chat

Utilisation : "&" + colorChar dans le chat

En utilisant le symbole '&' suivit du caractère de couleur, il est possible de formater son texte dans le chat.
Les caractères par défaut sont ceux de minecraft qui sont décrits ici.

Exemple :
    '&4rouge' donnera : 'rouge'
