Utilisation de la commande gteams du plugin GamingBlockPlug

Usage : /gteams list
              | /gteams add|remove <grade>
              | /gteams option <grade> <optionName> [value]
              | /gteams join <grade> [player]
              | /gteams leave [player];

Cette commande permet gerer tout les grades et leurs options.
Il est possible d'ajouter, d'enlever des grades mais aussi de changer plusieurs options.
Les options sont :
    -  Color => la couleur du grade dans le chat
    -  DisplayName => le nom du grade dans le chat
    -  MaxHomes => le nombre maximum de 'home'
    -  CanUseFly => Vrai/Faux > possibilité d'utiliser la commande /fly
    -  CanUseTpa => Vrai/Faux > possibilité d'utiliser la commande /gtpa
    -  CanUseWorld => Vrai/Faux > possibilité d'utiliser la commande /world
    -  CanUseSpectate => Vrai/Faux > possibilité d'utiliser la commande /spectate
    -  CanOpenShops => Vrai/Faux > possibilité d'ouvrir les 'shops' des personnes du même grade
    -  Salary => le salaire par semaine

Exemple :
    '/gteams add admin'
    Le grade admin est créé avec les droits par defauts sauf si il existe déja.
    '/gteams option admin DisplayName Test'
    Ceux sui possèdent le grade 'admin' auront le mot 'Test' à coté du pseudo.
    '/gteams join admin wormsor'
    Le joueur 'wormsor' possède désormais le grade 'admin'.
