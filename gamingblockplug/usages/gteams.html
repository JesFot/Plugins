Utilisation de la commande gteams du plugin GamingBlockPlug

Usage : /gteams list <[-|&]permission>

Cette commande permet gerer tout les grades et leurs options.
Il est possible d'ajouter, d'enlever des grades mais aussi de changer plusieurs options.

Exemple :
    '/gteams add admin'
    Le grade admin est créé avec les droits par defauts sauf si il existe déja.
    '/gteams option admin DisplayName Test'
    Ceux sui possèdent le grade 'admin' auront le mot 'Test' à coté du pseudo.
    '/gteams join admin wormsor'
    Le joueur 'wormsor' possède désormais le grade 'admin'.
Version d'ajout: Version 1.20
Dernière mise-à-jour: Version 2.0
