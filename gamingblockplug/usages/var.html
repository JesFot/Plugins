Utilisation de la commande var du plugin Eliryum

Usage: /var <varName> or /var set <varName> <type> <value> or /var unset <varName>

Cette commande permet de voir le contenu, définir, modifier ou supprimer une variable.
En utilisant '/var <varName>' vous pouvez voir le contenu de la variable 'varName' qui s'affichera dans le chat.
Si la variable n'est pas définie, un 'null' vous seras envoyé.

En utilisant '/var set <varName> ...' vous pouvez définir ou modifier une variable.
le 'varName' correspond au nom de la variable; 'type' correspond au type de variable :
    -  'string' -> chaine de texte (ou simplement du texte),
    -  'integer' -> nombre entier (comme -25; 0; 100 -> nombres sans virgules),
    -  'boolean' -> vraie ou faux (accepte 'true'(1) et 'false'(0)),
    -  'float' -> nombre (avec ou sans virgules).
le 'value' correspond donc à la valeur que possèderas la variable.

En utilisant '/var unset <varName>' vous pouvez supprimer la variable 'varName'.


Exemple :
    '/var plugin' va retourner :
    'Value: true'
    '/var test string une chaine de characteres...' enregistre la variable 'test'
    '/var test' va donc retourner :
    'Value: une chaine de characteres...'
    Pour la supprimer :
    '/var unset test'
    '/var test' retournera :
    'Value: null'

à voir : Utilisation des varibles de le chat.

Permissions :
'/var <varName>' -> "GamingBlockPlug.var.view" -> defaut : tout le monde.
'/var set|unset ...' -> "GamingBlockPlug.var.set" -> defaut : admins.
