Utilisation de la commande testgbp du plugin GamingBlockPlug

Usage : /testgbp [word]

Cette commande sert juste à tester le fonctionnement du plugin.
Si le plugin est actif vous voyez dans le chat : "Cette commande et donc le plugin fonctionnent."
Et si vous donner un 'word' (mot) vous voyez : "Cette commande et donc le plugin fonctionnent.Et votre texte : [word]".
