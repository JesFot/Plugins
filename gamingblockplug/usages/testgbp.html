Utilisation de la commande testgbp du plugin GamingBlockPlug

Usage : /testgbp [word]

Cette commande sert juste à tester le fonctionnement du plugin.
Si le plugin est actif : "Cette commande et donc le plugin fonctionnent."
Si vous donner un 'word' (mot) : "Cette commande et donc le plugin fonctionnent.Et votre texte : [word]".
