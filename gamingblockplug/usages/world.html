Utilisation de la commande world du plugin GamingBlockPlug

Usage : /world list
              | /world tp <world> [Player]
              | /world set <world> <load|unload> [seed] [env] [type]
              | /world set <world> options <GameMode|Group|...> <value>

Cette commande sert à se teleporter entre les mondes où à les contrôler.
Elle n'est cependant pas utilisable en Gamemode aventure.
Elle permet de charger ou de décharger un monde, ou encore de changer des options listés ci-dessous.
Pour creer un nouveau monde, il suffit de faire comme si on chargait un monde existant (Il est cependant possible de donner la graine et le type de monde.

Liste des option:
  • GameMode (valeurs: 'Creative', 'Survival', 'Adventure', 'Spectator')
  • Group - Permet de définir le groupe du monde
  • KeepInventory - Permet de garder son inventaire en entrant dans le monde (seulement si dans même groupe)
  • KeepLastLocation - Permet de retourner à l'endroit où on était en qittant le monde (seulement si dans même groupe)
  • ChangeSpawnpoint - Permet de change le lieux de re-spawn (actif si pas dans même groupe)
  • AutoUnload - Permet le déchargement automatique du monde quand il est vide
Exemple :
    '/world set example load'
    '/world tp example'
    Charge le monde 'example' ou le cré si inexistant puis se teleporte dans ce monde.
    Pour changer le groupe de monde puis le décharger :
    '/world set example options Group group1'
    '/world set exmaple unload'
