Description de l'ajout de variables dans le chat

Utilisation: ${varName} dans le chat

En écrivant '${varName}' dans le chat avec 'varName' le nom de la variable désiré, toute cette partie sera transformé par
le contenu de la variable en question.
Ceci est appliquable quelque soit la position du '${varName}' das le message.
NB: L'enregistrement de couleurs est possible.
Exemple :
    'mon plugin est on : ${plugin}' donera dans le chat :
    'mon plugin est on : true'.
    'je fais d${plugin}es tests' donera dans le chat :
    'je fais dtruees tests'.

à voir : Commande /var, permet de voir et/ou modifier ces variables.

Permission :
'${varName}' -> "GamingBlockPlug.var.use" -> defaut : tout le monde.
