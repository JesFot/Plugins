MUtilisation de la commande testmpp du plugin MetalPonyPlug

Usage : /testmpp [word]

Cette commande sert juste à tester le fonctionnement du plugin.
Si le plugin est actif : "Cette commande et donc le plugin fonctionnent."
Si vous donner un 'word' (mot) : "Cette commande et donc le plugin fonctionnent.Et votre texte : [word]".

Si le mot est 'setting' et l'argument qui suit est 'set' then
Et encore que le 3eme est 'affich' comme '/testmpp setting set affich ' :
  Définie la classe (métier) qui doit être monté.

Si le 3eme argument est 'affichi' comme '/testmpp setting set affichi ' :
  Définie la classe (métier) qui doit être monté par l'Id de la classe.
