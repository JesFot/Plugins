MetalPonyPlug testmpp command usage

Usage : /testmpp [word]

This command is for test the plugin activation othin else.
If the plugin is active return : "Cette commande et donc le plugin fonctionnent."
If you give a word return : "Cette commande et donc le plugin fonctionnent.Et votre texte : [word]".

If the word is 'setting' and the argument after word is 'set' then
if the third argument is 'affich' like '/testmpp setting set affich ' :
  set Rank (Class) to be showed.

if the third argument id 'affichi' like '/testmpp setting set affichi ' :
  set Rank (Class) to be showed by the ID of the class.
