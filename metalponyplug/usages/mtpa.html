MetalPonyPlug mtpa command usage

Usage : /mtpa <player> [target]

This command looks like the essential's tpa command.
It's used to teleport to the player or teleport the player to an other 'target'.
only if the target say 'yes' (can say y | yes | n | non)

Example :
    '/mtpa lydia_drew'
    Now lydia_drew see :
    '[the player to be tp] wants to teleport to you, accepte ? [y/n]'
    lydia_drew write just 'y' or 'n', no '/'
Aliases : '/tpa'
