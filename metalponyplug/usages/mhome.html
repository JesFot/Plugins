MetalPonyPlug mhome command usage

Usage : /mhome [homeName] [set]

This command looks like the essential's home command.
It's used to set a 'home' (waypoint) with or without a name.
The 'homeName' is not necessary but to store more than one home it is needed.

Example :
    '/mhome set'
    '/mhome house set'
    Now there is 2 location stored : the base and 'house'.
    To go at one of these homes :
    '/mhome'
    '/mhome house'
Aliases : '/home' and '/h'
