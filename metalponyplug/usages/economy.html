Utilisation de la commande economy du plugin MetalPonyPlug

Usage : /economy [pay <player> <montant>]

Le texte sera couleur or, les liens en bleu souligné.
Vous pouvez ajouter " n " pour mettre une nouvelle ligne.

exemple :
    '/logmessage set une ligne n une deuxieme ligne n httpUnlien'
